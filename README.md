## Проектная работа «Сложно сосредоточиться»

### Цель работы:  
Сверстать адаптивный сайт с поддержкой светлой и тёмной тем, настроить автоматическое переключение между темами в зависимости от системных настроек пользователя, реализовав Pixel Perfect вёрстку по макету.

### Стек технологий
- HTML5 (семантическая разметка)
- CSS3 (Grid, Flexbox, CSS-переменные, медиазапросы)
- Mobile First подход
- Адаптивная вёрстка (мобильные устройства, планшеты, десктоп)
- CSS Custom Properties (переменные для тем)
- JavaScript (готовый скрипт переключения тем)

### Ключевые особенности и требования
Поддержка двух цветовых тем (светлая/тёмная)  
Автоматическое определение темы из системных настроек  
Принудительное переключение темы через кнопки  
Адаптивный дизайн для всех устройств  
Резиновая вёрстка без горизонтального скролла 
Mobile First подход  
Логические свойства CSS (margin-block-start и т.д.)  
Гриды для основной раскладки  
Без горизонтального скролла  
Доступность (aria-атрибуты, семантика)  
Оптимизация изображений (lazy loading, alt-тексты)  

### Результат
Полностью адаптивный сайт с бесшовным переключением между светлой и тёмной темами, корректно отображающийся на всех устройствах и учитывающий пользовательские предпочтения.

### Основные этапы
#### Подготовка и анализ
Изучение макетов для светлой и тёмной тем  
Выбор подхода: Mobile First  
Определение основной темы (рекомендуется тёмная)  
Анализ готовой HTML-разметки и JavaScript  

#### Базовая настройка
Подключение CSS-файлов в правильном порядке  
Настройка метатега color-scheme  
Подключение фавиконок для всех устройств  
Создание фиксированного фонового изображения  

#### Верстка статичной версии
Шапка сайта  
Семантические секции  
Карточки с текстовой информацией  
Стилизованные ссылки с состояниями  
Галерея (оптимизированные изображения с lazy loading, грид-раскладка для адаптивности)
Футер

#### Адаптивность
Медиазапросы для планшетов (≥768px)  
Медиазапросы для десктопов (≥1024px)   
Плавное масштабирование элементов  
Сохранение пропорций на всех разрешениях  

#### Цветовые темы
Создание CSS-переменных в variables.css  
Файлы light.css и dark.css для переопределения  
Поддержка системных предпочтений через prefers-color-scheme  
Принудительное переключение через JavaScript-классы  

#### Интерактивные элементы
Стилизация кнопок переключения тем  
Состояния :hover, :active, :focus-visible
Активная кнопка с pointer-events: none
